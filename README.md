
MichinDelicatessen Restaurant es una aplicación web diseñada para optimizar el proceso de toma de 
pedidos en un restaurante de alta cocina. Su interfaz gráfica permite registrar la información del cliente, 
seleccionar productos organizados por categorías (sopas, platos principales, bebidas, postres y adicionales) 
y visualizar un resumen dinámico del pedido antes de confirmar la compra.

La aplicación calcula automáticamente el costo de cada producto y el valor total del pedido, registra la fecha y hora 
de la solicitud, almacena el historial de pedidos mediante LocalStorage y permite consultar o administrar los pedidos 
realizados sin necesidad de una base de datos externa.

Su diseño moderno, con imágenes, efectos visuales y una experiencia de usuario intuitiva, facilita el proceso de compra 
y mejora la interacción entre el cliente y el restaurante.

Características principales
🐱 Diseño personalizado con la identidad gráfica de Michin Delicatessen.
👤 Registro de datos del cliente.
🍲 Selección de productos por categorías.
💲 Visualización del precio de cada producto.
🧾 Resumen del pedido en tiempo real.
🧮 Cálculo automático del total a pagar.
📅 Registro de fecha y hora del pedido.
💾 Almacenamiento del historial mediante LocalStorage.
📋 Consulta de pedidos realizados.
🗑️ Eliminación de pedidos individuales o del historial completo.
📱 Diseño adaptable y fácil de utilizar.

Objetivo de la aplicación
El objetivo de Michin Delicatessen es ofrecer una solución digital que agilice el proceso de registro y gestión de pedidos 
en un restaurante, reduciendo errores en la toma de órdenes, automatizando el cálculo de costos y proporcionando una experiencia 
de usuario moderna, organizada y eficiente tanto para los clientes como para el personal del establecimiento.
